iPhone X

一直以来，我们都心存一个设想，期待着能够打造出这样一部 iPhone： 它有整面的屏幕，能让你在使用时完全沉浸其中，仿佛忘记了它的存在。它是如此智能，你的一触、一碰、一言、一语，哪怕是轻轻一瞥， 都会得到它心有灵犀的回应。而这个设想，终于随着 iPhone X 的到来成为了现实。现在，就跟未来见个面吧。

设计和显示屏

全面屏，全面绽放。

超视网膜显示屏

整部 iPhone X，看起来就是一块几乎整面的屏幕。全新的 5.8 英寸超视网膜显示屏用起来倍感称手，看起来绚丽夺目。

创新科技

显示屏采用新的工艺和技术，精准贴合机身的曲线，一直延伸至优雅圆润的边角。

全新超视网膜显示屏，精工细造，精美绽放。

超视网膜高清显示屏采用的 OLED 显示面板，远非普通的 OLED。它按照我们的严苛标准量身打造，拥有 1,000,000:1 的对比度、高分辨率和出色的亮度、广色域支持，以及出众的色彩精准度，能够实现 HDR 显示。

原深感摄像头

小小的空间内，遍布着我们潜心研发的各种先进科技，包括可实现面容 ID 功能的各种镜头和感应器。

防溅抗水，还防尘。

我们深入微观层面对 iPhone X 进行了精密设计，以实现防溅、抗水和防尘性能。

简单自然的手势

熟悉的手势，使用起来自然流畅、简单直观。无论你在哪个界面，只需轻扫一下就能返回主屏幕，不必再使用按钮。

识别技术，从此面貌一新。

你的脸，现在就是你的密码。面容 ID 功能为设备解锁、身份验证和支付带来了一种安全可靠的新方式。

面谱绘制

面容 ID 功能通过原深感摄像头来实现，设置起来也非常简单。它会投射超过 30,000 个肉眼不可见的光点，并对它们进行分析，为你的脸部绘制精确细致的深度图。

Apple Pay

面容 ID 非常安全，你可以放心将它用于 Apple Pay。只需看一眼手机，就能轻松进行支付。

你的脸，就是一张安全的通行证。

有了面容 ID 的帮助，iPhone X 可以在你注视它时才会解锁。而独到的设计，让它可以防止被照片或面具假冒的人脸欺骗。你的面谱数据会被加密并保护在安全隔区中。身份验证会在设备上即时进行，不会在云端处理。

一旦认识你，就总能认出你。

A11 仿生这款芯片能够利用机器学习技术来识别你样貌的变化。当你戴上眼镜、帽子或留起胡须时，你的朋友或许一眼认不出你，但是 iPhone X 可以。

善解人意，在你需要时才启动。

从进一步的安全层面考虑，我们为面容 ID 增加了注视感知功能，也就是说，只有当你睁眼注视着 iPhoneX 时，它才会解锁。另外，面容 ID 功能也可用来显示通知和信息，在你阅读内容时保持屏幕常亮，或调低闹钟或铃声的音量。

动话表情

原深感摄像头能对 50 多种不同的面部肌肉运动进行分析，从而以 12 款“动话表情”镜像出你的神态。快来化身为熊猫、兔子或机器人，展现一个真实的你吧。

摄影这门艺术，

说高深，也简单。

拥有比以往尺寸更大、速度更快的 1200 万像素感光元件、新的色彩滤镜、更深层的像素，以及新的支持光学图像防抖的长焦镜头。

人像光效

深度感应摄像头和精确细致的面谱绘制技术，营造出媲美影棚的惊艳光效。

光学变焦

iPhone X 能通过广角和长焦镜头实现光学变焦。另外，通过数码变焦，还能拍摄最高达 10 倍变焦的照片和最高达 6 倍变焦的视频。

超乎想象的强劲动力。

A11 仿生拥有六个核心和 43 亿个晶体管，与 A10 Fusion 芯片相比，四个能效核心的速度提升最高可达 70%，两个性能核心的速度提升最高可达 25% 之多。当需要提升处理速度时，中央处理器甚至能同时发挥全部六个核心的性能。

高效用电

第二代性能控制器和量身定制的电池设计，使充满电后的电池续航时间比 iPhone 7 最多可延长两个小时。

增强现实

A11 仿生能为游戏和 app 带来非同寻常的增强现实体验。

无线世界，无线充电。

iPhone X 专为将来的无线世界而设计，让你可以不被充电线左右。

开创性的 iOS，

为开创性的 iPhone 而生。

面对一部采用全面屏的手机，我们需要彻底重新思考和设计它的操作系统，并加入新的功能和手势。

iOS 11 的新功能

你可以在信息中发送“动话表情”。另外，Siri 又多了一个新的身份：你的私人 DJ。

增强现实

在全球范围内，iOS 开创了一个广阔的增强现实平台，令你能尽情体验让人眼界大开的增强现实类游戏和 app。
